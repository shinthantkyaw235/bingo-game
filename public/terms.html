MABA Myanmar- Japan Festival Bingo Game

စည်းမျဉ်းစည်းကမ်းများနှင့် သတ်မှတ်ချက်များ (Terms & Conditions)

MABA Myanmar- Japan Festival Bingo Game တွင် ပါဝင်ဆင်နွှဲရန်အတွက် အောက်ပါ စည်းမျဉ်းစည်းကမ်းများနှင့် သတ်မှတ်ချက်များကို သေချာစွာ ဖတ်ရှုပြီး သဘောတူညီရန် လိုအပ်ပါသည်။ သင်သည် စနစ်အတွင်းသို့ Login ဝင်ရောက်ခြင်း (သို့မဟုတ်) မှတ်ပုံတင်ခြင်းဖြင့် ဤစည်းကမ်းချက်များကို လိုက်နာရန် သဘောတူညီပြီး ဖြစ်သည်ဟု မှတ်ယူပါမည်။

၁။ အကောင့်ဖွင့်လှစ်ခြင်းနှင့် ကိုယ်ရေးအချက်အလက် (Registration & Account)

ဂိမ်းတွင် ပါဝင်ရန်အတွက် မိမိ၏ အမည်ရင်း၊ တိကျမှန်ကန်သော ဖုန်းနံပါတ် စသည်တို့နှင့်အတူ အကောင့်ဖွင့်လှစ်ခြင်း(Register) အား ဦးစွာပထမ ပြုလုပ်ရပါမည်။

လူတစ်ဦးလျှင် အကောင့် (၁) ခုသာ ဖွင့်လှစ်ခွင့်ရှိပြီး၊ အချက်အလက်အမှားများ ဖြည့်သွင်းထားပါက ကံစမ်းမဲပေါက်သော်လည်း ဆုထုတ်ယူခွင့်ကို ငြင်းပယ်ပိုင်ခွင့် ရှိသည်။

၂။ ဂိမ်းကစားခြင်းဆိုင်ရာ စည်းကမ်းချက်များ (Game Mechanics)

Bingo Game ကံစမ်းခွင့်ရရှိရန် ပွဲတော်အတွင်းရှိ ဆိုင်ခန်း (၁၀၀) အနက်မှ မိမိနှစ်သက်ရာ မတူညီသော ဆိုင်ခန်း (၁၀) ခန်း သို့ လူကိုယ်တိုင် သွားရောက်လေ့လာရပါမည်။

ဆိုင်ခန်းတစ်ခုချင်းစီတွင် ဆိုင်ရှင်မှ ပေးသော Code ကို မိမိဖုန်းဖြင့် ရိုက်ထည့်၍ Stamp ရယူရပါမည်။ ၎င်း Code တစ်ခုအား တစ်ကြိမ်သာ သုံးစွဲခွင့်ရှိမည်။ Game Road Map ပေါ်ရှိ Stamp တစ်ခုချင်းစီတွင် မိမိရရှိထားသောကုဒ်ကို ရိုက်သွင်းရပါမည်။

ဆိုင်ခန်းတစ်ခုချင်းစီတွင် သတ်မှတ်ထားသော Code အရေအတွက် (၁၀၀) သာ ရှိမည်ဖြစ်ပြီး၊ ကုန်ဆုံးသွားပါက ထိုဆိုင်ခန်းမှ Stamp ထပ်မံရရှိနိုင်တော့မည် မဟုတ်ပါ။

၃။ ဂိမ်းအချိန်အပိုင်းအခြားနှင့် Stamp သယ်ဆောင်ခွင့် (Sessions & Carry-over)

ဂိမ်းကို သတ်မှတ်ထားသော အချိန်အပိုင်းအခြား (Game Sessions) များအတိုင်းသာ ကစားခွင့်ရှိသည်။

အကယ်၍ လက်ရှိ Session အတွင်း Stamp (၁၀) ခု မပြည့်မီ အချိန်ပြည့်သွားပါက မိမိစုဆောင်းထားသော Stamp များသည် ပျက်ပြယ်မသွားဘဲ နောက်လာမည့် Session တွင် ဆက်လက်အကျုံးဝင်မည် ဖြစ်သည်။

သို့သော် သွားရောက်ပြီးသား ဆိုင်ခန်းများမှ Stamp ကို နောက်တစ်ကြိမ် ထပ်မံရယူခွင့် မရှိဘဲ၊ မသွားရသေးသော ဆိုင်ခန်းအသစ်များမှသာ ထပ်မံစုဆောင်းရပါမည်။

၄။ ဆုမဲများနှင့် ထုတ်ယူခြင်း (Prizes & Redemption)

Game Session တစ်ခုစီတွင် Stamp ၁၀ ခု အမြန်ဆုံးပြည့်ပြီး Bingo တန်းသူ ဦးရေ (၂၀) သာ Spin Wheel လှည့်၍ ဗလာမပါ ကံစမ်းခွင့် ရရှိမည်ဖြစ်သည်။

ဆုမဲပေါက်သူများသည် မိမိတို့ရရှိသော "ဆုမဲကုဒ် (Claim Code)" ကို ဖုန်းတွင် ပြသ၍ ပွဲတော်အတွင်းရှိ MABA ရုံးချုပ် (HQ) ကောင်တာ တွင် လူကိုယ်တိုင် သွားရောက်ထုတ်ယူရပါမည်။

ဆုထုတ်ယူချိန်တွင် စနစ်၌ မှတ်ပုံတင်ထားသော အမည်နှင့် ဖုန်းနံပါတ်တို့ကို ဝန်ထမ်းများထံ တိကျစွာ ပြောကြား ပြသရပါမည်။

၅။ မသမာမှုတားဆီးရေးနှင့် ပယ်ဖျက်ပိုင်ခွင့် (Anti-Fraud & Disqualification)

နည်းပညာကို အသုံးပြု၍ စနစ်အား တိုက်ခိုက်ခြင်း၊ ကလိမ်ကကျစ်ပြုလုပ်ခြင်း၊ Code စနစ်ကို လိမ်လည်ရန် ကြိုးပမ်းခြင်း စသည့် မသမာမှုတစ်စုံတစ်ရာ တွေ့ရှိပါက ၎င်း User အကောင့်အား မည်သည့်သတိပေးချက်မျှမရှိဘဲ ချက်ချင်းပိတ်သိမ်းမည်ဖြစ်ပြီး ရရှိထားသော ဆုမဲအားလုံးကို ပယ်ဖျက်ပါမည်။

အငြင်းပွားဖွယ်ရာ ကိစ္စရပ်များ ပေါ်ပေါက်လာပါက MABA ပွဲစီစဉ်သူကော်မတီ၏ ဆုံးဖြတ်ချက်သာ အတည် ဖြစ်သည်။

၆။ ကိုယ်ရေးအချက်အလက် ထိန်းသိမ်းမှု (Privacy Policy)

သင်ဖြည့်သွင်းထားသော အချက်အလက်များကို MABA မှ စနစ်တကျ ထိန်းသိမ်းထားမည်ဖြစ်ပြီး၊ ပွဲတော်ဆိုင်ရာ သတင်းအချက်အလက်များနှင့် နောင်ကျင်းပမည့် လှုပ်ရှားမှုများအတွက်သာ အသုံးပြုသွားမည်ဖြစ်ပါသည်။
