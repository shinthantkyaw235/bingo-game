🎉 MABA Smart Bingo Game မှ ကြိုဆိုပါတယ်! 🎉

ပွဲတော် (၂) ရက်တာ ကာလအတွင်း ဧည့်သည်များအတွက် ဆုမဲပေါင်းများစွာ ပါဝင်သည့် Bingo Game အစီအစဉ်ကို တစ်နေ့လျှင် (၃) ကြိမ်နှုန်းဖြင့် စုစုပေါင်း (၆) ကြိမ် ခွဲခြားကျင်းပသွားမည် ဖြစ်ပါသည်။

🔥 Grand Final သို့ တက်လှမ်းပါ! နောက်ဆုံးပိတ် Game Session (၆) သည် အဖိုးတန် Grand Prize (ဆုကြီးများ) ချီးမြှင့်မည့် အထူးပွဲစဉ်ဖြစ်ပါသည်။

စည်းမျဉ်း - အဆိုပါ Grand Final သို့ ဝင်ရောက်ကစားလိုပါက Session (၁) မှ (၅) အနက် တစ်ခုခုတွင် အနည်းဆုံး (၁) ကြိမ် ဝင်ရောက်ကစားဖူးသူ ဖြစ်ရပါမည်။

ဆုကြီးများကို ရယူရန် ပွဲအစကတည်းက လက်မနှေးဖို့တော့ လိုမယ်နော်။ ဂိမ်းကစားရန် နေရာအကန့်အသတ်ရှိသဖြင့် အောက်ပါလင့်ခ်တွင် အမြန်ဆုံး ကြိုတင်စာရင်းသွင်း (Register) ထားရန် ဖိတ်ခေါ်အပ်ပါသည်။
